# sassy-lib
## Custom SASS library
- Written in non-indented SASS (SCSS)
- In development
- Offers basic utilities (Will be updated)

*** Use Gulp to watch recompile SASS to css and debug (gulpfile.js is required)
```
npm i gulp gulp-sass sass gulp-purgecss --save-dev
```

  "devDependencies": {
